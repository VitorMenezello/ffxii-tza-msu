Installation and Usage

Hello there! This is an MSU-1 pack for The Legend of Zelda: A Link to the Past Randomizer with music from Final Fantasy XII - The Zodiac Age and extended MSU support. If you are completely new to using MSU-1 packs, you should first read qwertymodo's guide. Otherwise, follow these simple steps:

Download the RAR file by clicking the button below:
Download
Extract the files and place your ROM in the same folder with the name ffxii_tza_msu.sfc
Open your emulator of choice, load the ROM and play!
